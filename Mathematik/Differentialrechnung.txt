====== Differentialrechnung ======
Created Friday 11 September 2015

Grenzwerte und Grenzwerte von Funktionswerten

<Xn> = <X1, X2, X3, X4, ... > bezeichnet eine Folge von Zahlen

<Xn> = <1/n> = <1, 1/2, 1/3, 1/4, ... >

<Xn> = <(n+1)/n> = <2; 3/2; 4/3; 5/4; ... >

<Xn> = <(n-1)/n> = <0/1; 1/2; 2/3; ... >

<Xn> = <1> = <1; 1; 1; ...>
